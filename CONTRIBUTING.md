# Contributing to Windows 10 Longevity Suite

Thank you for considering contributing! 

## How to Contribute
1. **Report Bugs:** Open an issue describing the bug and your environment.
2. **Feature Requests:** Open an issue to discuss new script ideas.
3. **Pull Requests:** 
    - Fork the repo.
    - Create a feature branch.
    - Ensure your script includes logging and proper error handling (try/catch).
    - Submit a PR with a clear description of changes.

Please ensure all scripts are tested in a non-production environment first.
